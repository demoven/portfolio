<div data-lang-content="en">

**Human Activity Recognition (HAR) Engine** is a machine learning-driven analytical system designed to classify complex physical activities using high-dimensional sensory data. Developed as a core component of my technical portfolio, this project focuses on the intersection of **Signal Processing** and **Supervised Learning** to interpret data from smartphone accelerometers and gyroscopes.

I engineered a production-grade data pipeline using **Scikit-Learn**, integrating automated preprocessing stages such as **SimpleImputer** for data integrity. This modular architecture ensures the system is algorithm-agnostic, allowing for seamless transitions between different classification models. I conducted extensive benchmarking between **Random Forest**, **Decision Trees**, and **XGBoost**, utilizing **GridSearchCV** to fine-tune hyperparameters and optimize the bias-variance tradeoff.

By achieving high-precision classification across 6 distinct activity types (e.g., walking, climbing stairs, sitting), this project demonstrates my ability to build scalable ML workflows. It showcases my proficiency in **Pythonic data structures**, statistical validation through cross-validation, and the implementation of automated pipelines—essential skills for developing modern, data-intensive software solutions.

---

### Key Technical Achievements:
* **Robust Pipeline Engineering:** Implemented a full `sklearn.pipeline` to handle the end-to-end workflow from raw data to prediction.
* **Hyperparameter Optimization:** Leveraged `GridSearchCV` to systematically explore parameter spaces for XGBoost and Random Forest.
* **Performance Analysis:** Generated detailed classification reports and confusion matrices to validate model reliability on a 10,000+ observation dataset.
* **Technologies:** Python (Pandas, NumPy, Scikit-Learn, XGBoost), Matplotlib, Colab.

**Human Activity Recognition (HAR) Engine** est un système analytique basé sur l'apprentissage automatique, conçu pour classifier des activités physiques complexes à partir de données sensorielles de haute dimension. Développé comme une pièce maîtresse de mon portfolio technique, ce projet se concentre sur l'intersection du **traitement du signal** et de l'**apprentissage supervisé** pour interpréter les données provenant des accéléromètres et gyroscopes de smartphones.

J'ai conçu un pipeline de données de niveau production à l'aide de **Scikit-Learn**, intégrant des étapes de prétraitement automatisées telles que **StandardScaler** pour la normalisation des caractéristiques et **SimpleImputer** pour garantir l'intégrité des données. Cette architecture modulaire rend le système indépendant de l'algorithme choisi (*algorithm-agnostic*), permettant une transition fluide entre différents modèles de classification. J'ai réalisé un benchmarking approfondi entre les modèles **Random Forest**, **Decision Trees** et **XGBoost**, en utilisant **GridSearchCV** pour optimiser les hyperparamètres et équilibrer le compromis biais-variance.

En atteignant une classification de haute précision sur 6 types d'activités distinctes (ex: marche, montée d'escaliers, assis), ce projet démontre ma capacité à construire des flux de travail ML évolutifs. Il met en avant ma maîtrise des **structures de données Python**, la validation statistique par validation croisée et la mise en œuvre de pipelines automatisés — des compétences essentielles pour le développement de solutions logicielles modernes axées sur les données.

---

### Réalisations Techniques Clés :
* **Ingénierie de Pipeline Robuste :** Mise en œuvre d'un `sklearn.pipeline` complet pour gérer le flux de travail de bout en bout, des données brutes à la prédiction.
* **Optimisation des Hyperparamètres :** Utilisation de `GridSearchCV` pour explorer systématiquement les espaces de paramètres de XGBoost et Random Forest.
* **Analyse de Performance :** Génération de rapports de classification détaillés et de matrices de confusion pour valider la fiabilité du modèle sur un jeu de données de plus de 10 000 observations.
* **Technologies :** Python (Pandas, NumPy, Scikit-Learn, XGBoost), Matplotlib, Colab.
