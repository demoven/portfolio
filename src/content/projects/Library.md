<div data-lang-content="en">

**Use Case Library** is a custom WordPress plugin developed during my academic exchange at **Windesheim University** in the Netherlands. This project was a collaborative effort within an international team of four—consisting of two Dutch students, one French peer, and myself—working directly for our client, **Rob Van De Star**. Together, we designed and implemented a professional solution to manage and showcase technical use cases for the Windesheim Technology Radar.

Developed as a full-stack WordPress solution, the plugin leverages a custom **MySQL database schema** to store and retrieve complex project data efficiently. The project highlights my ability to bridge client requirements with robust technical implementation in a multicultural environment.

The plugin provides a complete ecosystem for use case management, featuring:

* **Dynamic Library Display**: A user-friendly interface powered by **AJAX** to provide a seamless browsing experience. It includes advanced filtering systems that allow visitors to sort projects by **Innovation Sectors**, **Value Chain stages**, **Themes**, and **Sustainable Development Goals (SDGs)**.
* **Integrated Submission System**: A custom front-end form that enables users to submit new project data. This system features robust **PHP validation**, secure **image upload handling**, and a custom **REST API endpoint** to process submissions without reloading the page.
* **Automated Workflow**: To streamline communication, the plugin includes a dedicated mailer system that sends automated confirmations to contributors and notification alerts to administrators upon new submissions.
* **Bespoke Admin Dashboard**: A centralized management area built specifically for the client's workflow. It features a paginated and sortable list of all submissions, allowing administrators to review, edit, and publish use cases with a single click.

**Use Case Library** est un plugin WordPress personnalisé développé lors de mon échange académique à l'**Université Windesheim** aux Pays-Bas. Ce projet est le fruit d'un travail collaboratif au sein d'une équipe internationale de quatre personnes — deux étudiants néerlandais, un pair français et moi-même — travaillant directement pour notre client, **Rob Van De Star**. Ensemble, nous avons conçu et implémenté une solution professionnelle pour gérer et présenter des cas d'usage techniques pour le Windesheim Technology Radar.

Développé comme une solution WordPress full-stack, le plugin exploite un schéma de **base de données MySQL** personnalisé pour stocker et récupérer efficacement des données de projets complexes. Ce projet met en valeur ma capacité à faire le lien entre les exigences client et une implémentation technique robuste dans un environnement multiculturel.

Le plugin fournit un écosystème complet de gestion des cas d'usage, avec notamment :

* **Affichage dynamique de la bibliothèque** : Une interface conviviale propulsée par **AJAX** pour une navigation fluide. Elle comprend des systèmes de filtres avancés permettant de trier les projets par **Secteurs d'Innovation**, **étapes de la Chaîne de Valeur**, **Thèmes** et **Objectifs de Développement Durable (ODD)**.
* **Système de soumission intégré** : Un formulaire front-end personnalisé permettant aux utilisateurs de soumettre de nouveaux projets. Ce système dispose d'une **validation PHP** robuste, d'une gestion sécurisée de **l'upload d'images** et d'un **endpoint REST API** pour traiter les soumissions sans recharger la page.
* **Workflow automatisé** : Pour fluidifier la communication, le plugin inclut un système de messagerie dédié qui envoie des confirmations automatiques aux contributeurs et des alertes aux administrateurs lors de nouvelles soumissions.
* **Tableau de bord administrateur sur-mesure** : Un espace de gestion centralisé conçu spécifiquement pour le flux de travail du client, avec une liste paginée et triable de toutes les soumissions permettant de réviser, modifier et publier les cas d'usage en un clic.
